Hướng dẫn sử dụng nhanh
Bước 1: Tạo thư mục mới và tải các file cần thiết về
```
# Tạo thư mục
mkdir wordpress
cd wordpress

#Tải các file tài nguyên
git clone https://github.com/vanlong32/wordpress-nginx-mysql-docker.git

# Tạo container
docker-compose up
```

Bước 2: Upload code wordpress vào thư mục public_html

Truy cập 192.168.1.140 để vào trang web
